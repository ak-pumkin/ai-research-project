# AI Research Assistant with Google Multi-Agent Collaboration

## Description
This project is a multi-agent AI assistant that summarizes research articles, fact-checks statements, and generates Python code using **Google PaLM API** and **Custom Search API**. It demonstrates multi-agent collaboration, session/memory management, and Google-only tools.

## Features
- Multi-Agent System: Summarizer, Fact-Checker, Code Generator
- Google PaLM API for text/code generation
- Google Custom Search for fact-checking
- Session and long-term memory tracking
- Streamlit Web Interface
